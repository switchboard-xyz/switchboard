## Getting Started

To get started, clone the
[sbv2-aptos](https://github.com/switchboard-xyz/sbv2-aptos) repository.

```bash
git clone https://github.com/switchboard-xyz/sbv2-aptos
```

Then install the dependencies

```bash
cd sbv2-aptos
pnpm install
```
