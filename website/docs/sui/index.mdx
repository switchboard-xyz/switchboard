## Getting Started

To get started, clone the
[sbv2-sui](https://github.com/switchboard-xyz/sui-sdk) repository.

```bash
git clone https://github.com/switchboard-xyz/sui-sdk
```

Then install the dependencies

```bash
cd sbv2-sui
pnpm install
```
