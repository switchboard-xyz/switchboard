Liquidity

Switchboard oracles can be used to fetch order book depth and AMM liquidity.

Binance Liquidityâ€‹

note

This is not a full job definition. You should pair it with some conditional logic.

sol_binance_orderbook.json
{
  "tasks": [
    {
      "httpTask": {
        "url": "https://www.binance.com/api/v3/ticker/bookTicker?symbol=SOLUSDT",
        "method": "METHOD_GET"
      }
    },
    {
      "cacheTask": {
        "cacheItems": [
          {
            "variableName": "BINANCE_BID_QTY",
            "job": {
              "tasks": [
                {
                  "jsonParseTask": {
                    "path": "$.bidQty",
                    "aggregationMethod": "NONE"
                  }
                }
              ]
            }
          },
          {
            "variableName": "BINANCE_ASK_QTY",
            "job": {
              "tasks": [
                {
                  "jsonParseTask": {
                    "path": "$.askQty",
                    "aggregationMethod": "NONE"
                  }
                }
              ]
            }
          }
        ]
      }
    },
    {
      "valueTask": {
        "big": "${BINANCE_ASK_QTY}"
      }
    }
  ]
}
