publicstaticcreate

  • create(program: SwitchboardProgram, params: AggregatorHistoryInit, options?: any): Promise<[AggregatorHistoryBuffer, TransactionSignature]>

  • Create a history buffer for an aggregator and store the last N samples in a round robin history buffer.

    Parameters

    • program: SwitchboardProgram

      The SwitchboardProgram.

    • params: AggregatorHistoryInit

      history buffer configuration parameters.

    • optionaloptions: any

    Returns Promise<[AggregatorHistoryBuffer, TransactionSignature]>

    TransactionObject that will create the AggregatorHistoryBuffer.

    Basic usage example:

    import { AggregatorAccount,AggregatorHistoryBuffer } from '@switchboard-xyz/solana.js';
    const aggregatorAccount = new AggregatorAccount(program, aggregatorKey);
    const aggregator = await aggregatorAccount.loadData();
    const [historyBuffer, addHistorySignature] = await AggregatorHistoryBuffer.create(program, {
       aggregatorAccount,
       maxSamples: 10000,
    });
    const history = await historyBuffer.loadData();
