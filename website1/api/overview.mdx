## Setup

The [switchboard-xyz/sbv2-core](https://github.com/switchboard-xyz/sbv2-core)
repo contains the chain agnostic code such as the CLI and OracleJob protobufs
along with a git submodule for each chain specific SDK.

To get started, clone the sbv2-core repo along with the submodules.

```bash
git clone --recurse-submodules https://github.com/switchboard-xyz/sbv2-core.git
cd sbv2-core
```
